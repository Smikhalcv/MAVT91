Вы находитесь на главное странице сайта тестового задания для группы компаний Ask42.

В данном проекте реализованно FrontEnd часть по средствам HTML, CSS и JavaScript(Leaflet).

BackEnd часть предствлена языком Python 3 на основе фреймворка Django.

Цель проекта состоит в записи координат, указанных точек в бызу данных (PostgeSQL) и ихвыводе.

Во FrontEnd часть по средствам Ajax запроса осуществляется передача данных на сервер Django.

На серверной части - данные заносятся в базу данных и извлекаются для отображения.
